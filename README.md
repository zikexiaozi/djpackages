#### 使用方式：

1. 先cd进package目录，然后执行

```bash
 git clone https://github.com/zikexiaozi/djpackages
```
2. 或者添加下面代码到feeds.conf.default文件

```bash
 src-git djpackages https://github.com/zikexiaozi/djpackages
```
